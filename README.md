# Brussels Mongolia Site Demo

## Files

- `index.html` - main landing page
- `styles.css` - full visual system and responsive styles
- `script.js` - language toggle, reveal animation, reservation form behavior
- `proposal.md` - sales and pricing proposal for Brussels Mongolia
- `launch-package.md` - launch preparation summary
- `commercial-onepager.md` - client-facing short commercial proposal
- `domain-hosting-plan.md` - recommended domain and hosting setup
- `launch-checklist.md` - pre-launch and launch-day checklist
- `analytics-seo-checklist.md` - SEO and analytics setup checklist
- `robots.txt`, `sitemap.xml`, `site.webmanifest` - deployment support files
- `client-email-templates.md` - email outreach and follow-up templates
- `social-outreach.md` - WhatsApp / Instagram / Facebook outreach copy
- `pdf-proposal-copy.md` - clean proposal copy for PDF or doc
- `domain-decision-memo.md` - domain recommendation summary
- `closing-script.md` - closing language for meetings

## Demo use

Open `index.html` directly in a browser or serve the folder with a simple static server.
