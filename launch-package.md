# Brussels Mongolia Launch Package

## 1. Package goal

Энэ багцын зорилго нь concept site-г `launch-ready commercial package` болгон хувиргах. Өөрөөр хэлбэл:

- client-д үзүүлэхэд итгэл төрүүлэх
- production build эхлүүлэхэд шаардлагатай шийдвэрүүдийг тодорхой болгох
- launch day дээр алдаа багатай гарах

## 2. Package contents

- `index.html` - launch-ready landing page structure
- `styles.css` - responsive visual system
- `script.js` - bilingual switcher, CTA behavior, inquiry handoff
- `proposal.md` - proposal and pricing structure
- `sales-playbook.md` - outreach, meeting flow, objection handling
- `commercial-onepager.md` - client-facing short commercial summary
- `domain-hosting-plan.md` - domain, hosting, infrastructure recommendation
- `launch-checklist.md` - pre-launch and launch-day checklist
- `analytics-seo-checklist.md` - GA4, Search Console, SEO, conversion setup
- `robots.txt`, `sitemap.xml`, `site.webmanifest` - deployment support files

## 3. Production dependencies

Launch-аас өмнө client-оос заавал авах зүйлс:

- Final approved venue name format
- Final contact number(s)
- Reservation handoff owner
- Menu categories and signature items
- Real interior / food photos
- Social links
- Domain ownership decision

## 4. Recommended launch order

1. Domain and hosting шийдэх
2. Final content approval авах
3. Real media assets оруулах
4. Form handoff integration хийх
5. Analytics/Search Console суулгах
6. SEO metadata final болгох
7. Cross-device QA хийх
8. Launch хийх

## 5. What to say to client

`Энэ багц нь зөвхөн дизайн биш. Launch day хүртэл явах бүх гол шийдвэр, контент, tracking, domain, conversion flow-ийг нэг дор тодорхойлж өгч байна.`
