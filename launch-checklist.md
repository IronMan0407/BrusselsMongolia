# Launch Checklist

## Content

- Brand name spelling approved
- Contact numbers approved
- Address and map link confirmed
- Opening hours confirmed
- Menu categories approved
- Signature item names approved
- MN/EN copy approved

## Visuals

- Interior photos selected
- Food photos selected
- Hero image approved
- Logo / mark available
- Social profile links confirmed

## Technical

- Final domain registered
- DNS pointed correctly
- Hosting connected
- SSL active
- Canonical URL updated
- Robots and sitemap checked
- Manifest updated if icon added

## Conversion

- Reservation owner assigned
- Form destination tested
- Call CTA tested
- Maps CTA tested
- Mobile CTA placement checked

## Tracking

- GA4 installed
- Search Console verified
- Meta descriptions reviewed
- Open Graph fields updated
- Event tracking for CTA clicks configured

## QA

- iPhone / Android mobile check
- Desktop Chrome / Safari check
- Typography and spacing check
- Broken link check
- Form validation check
- Page speed quick review

## Launch Day

- Publish site
- Re-test CTA buttons
- Submit sitemap in Search Console
- Update Google Business / social bio with domain
- Monitor inquiries for first 7 days
